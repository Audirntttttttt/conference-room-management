# MeetingRoomManagementSystem

该会议室管理后台系统有以下功能：
* 查询当天、近七天、近七年访客与会议室使用数据
* 访客管理（访客信息、访客预约、访客访问）
* 会议室管理（会议室信息、会议室预约、会议室使用）
* 会议室与访客不同类型的统计分析
* 用户管理
* 操作日志

管理员账号：admin
管理员密码：admin-123

该系统使用了vue与layui框架进行搭建、使用E-charts进行统计图表的绘制
